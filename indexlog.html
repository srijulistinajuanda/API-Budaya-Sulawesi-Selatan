To get characters you need to make a GET call to the following url :

Search by word : http://localhost/budaya/API/search.php?q=[Searched Word]&cat=[Category]

QUERY PARAMETERS

Field Type Description
q String (optional) A search word to find character by name.
cat String Category (makanan, tari, pakaian, rumah).

QUERY HEADERS

Field Type Description
key String Your API key.

The Westeros API uses the following error codes:

Code Meaning
200 Request Valid
300 Parameter Invalid
400 Api Key/Token Not Valid secret_key This error appears if you use an unknow API key or if your API key expired.
